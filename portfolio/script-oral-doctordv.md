---

## Présentation courte

DoctoRDV est une application web prévue pour gérer des rendez-vous médicaux fictifs.

Le contexte utilisé est une maison médicale fictive appelée MediCentre Nova.

L’objectif est de remplacer une gestion principalement téléphonique par une solution plus organisée.

---

## Besoin identifié

Le besoin principal est de mieux gérer :

- les praticiens ;
- les spécialités ;
- les disponibilités ;
- les rendez-vous ;
- le planning des praticiens.

---

## Solution prévue

La solution prévue est une application web simple.

Elle permettra à un patient fictif de rechercher un praticien, consulter ses disponibilités et réserver un rendez-vous.

Elle permettra aussi au praticien de consulter son planning.
