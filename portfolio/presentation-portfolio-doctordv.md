# Portfolio — DoctoRDV

## Présentation du projet

DoctoRDV est un projet d’application web réalisé dans le cadre du BTS SIO option SLAM.

L’objectif est de préparer une application simple de gestion de rendez-vous médicaux fictifs.

Le contexte utilisé est une maison médicale fictive appelée MediCentre Nova.

---

## Contexte du projet

MediCentre Nova gère principalement ses rendez-vous par téléphone.

Cette organisation peut poser plusieurs problèmes :

- erreurs de planning ;
- oublis de rendez-vous ;
- difficulté à voir les disponibilités ;
- manque de suivi pour les praticiens.

---

## Objectif de l’application

L’objectif de DoctoRDV est de permettre une gestion plus claire des rendez-vous.

L’application devra permettre :

- la recherche d’un praticien ;
- la consultation d’une fiche praticien ;
- la consultation des disponibilités ;
- la réservation d’un rendez-vous ;
- la consultation du planning par un praticien ;
- une gestion simple par un administrateur.

---

## Périmètre du projet

Pour rester réaliste, la première version sera simple.

### Fonctions prévues

- gestion des utilisateurs ;
- rôles administrateur, praticien et patient ;
- gestion des praticiens ;
- gestion des spécialités ;
- gestion des disponibilités ;
- réservation de rendez-vous.

### Fonctions non prévues dans la première version

- paiement en ligne ;
- téléconsultation ;
- rappel SMS ;
- dossier médical ;
- ordonnance ;
- messagerie patient/praticien.
